ESModule
    方案一:
        导出: export let a = 1; 导出去的是标识符
        导入: import {a} from "./a.js"; 导入也是标识符
    方案二:
        导出: export { a }; 导出去的是标识符
        导入: import { a  } from "./a.js"; 导入也是标识符
    方案三(默认导出):
        特点:模块中，只有一个数据 a 导出的话用 export default
        导出: export default a; 导出去的是标识符
        导入: import a from "./a.js"; 导入也是标识符
