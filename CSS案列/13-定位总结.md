# 1.1 绝对定位居中算法

- ~~~ CSS
    <!-- 1.水平居中 -->
    /* 1.left 走50% 父容器宽度的一半 */
    left : 50%;
    /*2. margin 负值 往左走 自己盒子宽度的一半*/
    margin-left : 50%;
    <!-- 2.垂直居中 -->
    /* 1.top 走50% 父容器高度的一半 */
    top : 50%;
    /*2. margin 负值 往上走 自己盒子高度的一半*/
    margin-top : 50%;
  ~~~
